Build Your Own Voice Assistant with PowerShell 🎙️

Welcome back to techsavvyp.com!Today we’re doing something futuristic and fun: creating a mini voice assistant using PowerShell. That’s right—your computer will talk to you, greet you, and even deliver custom messages. No downloads, no AI models—just pure PowerShell magic!

🧠 What You’ll Learn

We’ll use the built-in System.Speech library to make your computer speak. You’ll be able to customize greetings, reminders, or even jokes. It’s perfect for kids, coders, or anyone who wants their PC to feel a little more alive.

🛠️ Step-by-Step Guide

Step 1: Open PowerShell (you don’t need admin mode for this one).

Step 2: Paste the code below into the PowerShell window.

But you might be wondering: “How do I make it greet me every time I open PowerShell?” Let’s turn that one-time script into a permanent companion by saving it to your PowerShell profile.

🧠 Step-by-Step Guide

1. Open PowerShell (regular mode is fine). 2. Type this command and press Enter: "notepad $PROFILE" 3. If the file doesn’t exist, PowerShell will ask to create it—click Yes.

4. If your proile is empty just paste the code,save and close notepad,if it's not empty paste the code below the already existing code: 

Add-Type -AssemblyName System.Speech
$speak = New-Object System.Speech.Synthesis.SpeechSynthesizer

$speak.Speak("Welcome back to techsavvyp dot com.")
$speak.Speak("Your next update is scheduled for September 26th, 2025.")
$speak.Speak("Stay curious, stay creative, and stay tech savvy!")

Just change the $speak.Speak() lines to match your theme! If you want,you can add some conditions like: 


if ((Get-Date).DayOfWeek -eq 'Friday') {
    $speak.Speak("Happy Friday! Time to wrap up the week.")
}
