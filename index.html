TELNET

🧠 What is Telnet?
Telnet is like your computer saying: “Let me sit at the other computer’s desk and type commands as if I’m right there!”

🧸 Why do we use it?
- To check if another computer is awake and listening
- To talk to servers (like mail or web servers) using secret codes
- To test if a port (a room number) is open

📬 Example:
You type: telnet google.com 80
Your computer tries to connect to Google’s web server on port 80 (the room for websites). If it works, you can type special messages and see how the server responds.

🕵️ But be careful!
Telnet is like sending messages on postcards—everyone can read them! That’s why people now use SSH, which is like sending messages in sealed envelopes ✉️🔒.
