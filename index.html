Build a Mood Terminal on Mac 🍎 (and Windows 🖥️) : Color, Sound, and Emoji Vibes!

Want your Mac to reflect your mood? This playful project transforms the Terminal into a vibe-matching experience using bash and AppleScript. Whether you're feeling joyful, sleepy, or just need a boost, your Terminal can now respond with colors, sounds, and emojis.

🎨 What You’ll Build

  • Color themes using ANSI escape codes
  • Voice feedback with say and AppleScript
  • Emoji prompts and ASCII art
  • Gentle reflection questions

How to do it:

Step 1

Launch TextEdit

Step 2

Go to Format > Make Plain Text (important!)

Step 3

Go to File > Save

Step 4

Name it: mood_terminal.command

Step 5

Save it to your desktop. Make sure the file format is set to plain text and hit save

Step 6

Open terminal and type:

chmod +x ~/Desktop/mood_terminal.command
Make sure to change the path( after ~/ )to wherever the file is located

Step 7

Run it

The magic code> 

#!/bin/bash

# Mood Terminal for macOS
clear
echo "🌈 Welcome to Mood Terminal!"
echo "Choose your mood:"
echo "1. 😊 Happy"
echo "2. 😐 Neutral"
echo "3. 😢 Sad"
echo "4. 😠 Angry"
echo "5. 😴 Sleepy"
read -p "Enter number (1–5): " mood

# Reset colors
RESET="\033[0m"

# Define mood actions
case $mood in
  1)
    echo -e "\033[1;33mYou chose HAPPY! 😊\033[0m"
    say "You're glowing today!"
    osascript -e 'display notification "Keep smiling!" with title "Mood Terminal"'
    ;;
  2)
    echo -e "\033[1;37mYou chose NEUTRAL 😐\033[0m"
    say "Just another day. You're doing great."
    osascript -e 'display notification "Steady and strong." with title "Mood Terminal"'
    ;;
  3)
    echo -e "\033[1;34mYou chose SAD 😢\033[0m"
    say "It's okay to feel this way."
    osascript -e 'display notification "Sending virtual hugs 💙" with title "Mood Terminal"'
    ;;
  4)
    echo -e "\033[1;31mYou chose ANGRY 😠\033[0m"
    say "Take a deep breath. You're powerful."
    osascript -e 'display notification "Let it out safely." with title "Mood Terminal"'
    ;;
  5)
    echo -e "\033[1;35mYou chose SLEEPY 😴\033[0m"
    say "Time to recharge. Sweet dreams!"
    osascript -e 'display notification "Rest is important." with title "Mood Terminal"'
    ;;
  *)
    echo -e "\033[1;30mInvalid choice. Try again.\033[0m"
    ;;
esac

echo ""
echo "📝 Reflection: What made you smile today?"
echo "💡 Tip: You can edit this script to add your own moods and messages!"

For Windows

🧠 What You’ll Learn

We’ll use the built-in color command in CMD to change the background and text color based on your mood. You’ll create a simple menu that lets you choose your vibe for the day.

🛠️ Step-by-Step Guide

Step 1: Open Notepad

Step 2: Paste the code below

Step 3: Save the file as mood.bat (not mood.bat.txt)

🎨 The CMD Code


@echo off
title Mood-Changing Command Prompt
:MENU
cls
echo ================================
echo     Choose Your Mood Today
echo ================================
echo.
echo 1. Chill 😎       (Blue + White)
echo 2. Focused 🔥     (Red + Yellow)
echo 3. Creative 🌈     (Purple + Aqua)
echo 4. Mysterious 🕵️  (Black + Green)
echo 5. Default 🖤      (Reset Colors)
echo 6. Exit 🚪
echo.
set /p mood=Enter your choice (1-6): 

if "%mood%"=="1" (
    color 1F
    echo Mood set to Chill 😎
    pause
    goto MENU
)
if "%mood%"=="2" (
    color 4E
    echo Mood set to Focused 🔥
    pause
    goto MENU
)
if "%mood%"=="3" (
    color D3
    echo Mood set to Creative 🌈
    pause
    goto MENU
)
if "%mood%"=="4" (
    color 0A
    echo Mood set to Mysterious 🕵️
    pause
    goto MENU
)
if "%mood%"=="5" (
    color 07
    echo Mood reset to Default 🖤
    pause
    goto MENU
)
if "%mood%"=="6" (
    echo Goodbye! Stay Tech Savvy 💡
    exit
)

echo Invalid choice. Please try again.
pause
goto MENU

🎮 What the Colors Mean

  • 1F → Blue background, bright white text (Chill)
  • 4E → Red background, yellow text (Focused)
  • D3 → Light purple background, aqua text (Creative)
  • 0A → Black background, green text (Mysterious)
  • 07 → Default CMD look (Reset)
