Daily Tech Quote Generator 💬

Welcome to the Tech-Savvy Projekt! Today’s idea is simple, inspiring, and perfect for beginners: a quote generator that delivers a random tech-related message every time you run it. Whether you’re starting your day, teaching kids, or just need a boost, this project adds a spark of positivity to your terminal.

🧠 What You’ll Learn

  • How to use random selection in CMD and PowerShell
  • How to display motivational quotes in the terminal
  • How to save and run scripts easily

🛠️ Step-by-Step Guide

Step 1: Choose your version—CMD or PowerShell.

Step 2: Open Notepad.

Step 3: Paste the code from your choosen version below.

Step 4: Save the file 1(CMD).As quotes.bat(make sure it is not quotes.bat.txt) 2(POWERSHELL).Save it as mquotes.ps1.You can save both to your Desktop.

Step 5: Run it and enjoy your daily dose of tech wisdom!

🖥️ CMD Version (Batch File)


@echo off
setlocal enabledelayedexpansion

set /a pick=%random%%%5

if !pick!==0 echo "Technology is best when it brings people together. – Matt Mullenweg"
if !pick!==1 echo "The best way to predict the future is to invent it. – Alan Kay"
if !pick!==2 echo "Code is like humor. When you have to explain it, it’s bad. – Cory House"
if !pick!==3 echo "Stay curious. Stay creative. Stay tech savvy!"
if !pick!==4 echo "Any sufficiently advanced technology is indistinguishable from magic. – Arthur C. Clarke"

pause

⚙️ PowerShell Version


$quotes = @(
  "Technology is best when it brings people together. – Matt Mullenweg",
  "The best way to predict the future is to invent it. – Alan Kay",
  "Code is like humor. When you have to explain it, it’s bad. – Cory House",
  "Stay curious. Stay creative. Stay tech savvy!",
  "Any sufficiently advanced technology is indistinguishable from magic. – Arthur C. Clarke"
)

$pick = Get-Random -Minimum 0 -Maximum $quotes.Count
Write-Host $quotes[$pick]
