Mood-Changing Command Prompt 🎨

Welcome back to the Tech-Savvy Projekt! Today we’re adding a splash of personality to your Command Prompt. What if your terminal could reflect your mood—chill, focused, creative, or chaotic? With a few lines of code, you can turn your CMD window into a digital mood ring. Let’s make your console feel alive!

🧠 What You’ll Learn

We’ll use the built-in color command in CMD to change the background and text color based on your mood. You’ll create a simple menu that lets you choose your vibe for the day.

🛠️ Step-by-Step Guide

Step 1: Open Notepad

Step 2: Paste the code below

Step 3: Save the file as mood.bat (not mood.bat.txt)

🎨 The CMD Code


@echo off
title Mood-Changing Command Prompt
:MENU
cls
echo ================================
echo     Choose Your Mood Today
echo ================================
echo.
echo 1. Chill 😎       (Blue + White)
echo 2. Focused 🔥     (Red + Yellow)
echo 3. Creative 🌈     (Purple + Aqua)
echo 4. Mysterious 🕵️  (Black + Green)
echo 5. Default 🖤      (Reset Colors)
echo 6. Exit 🚪
echo.
set /p mood=Enter your choice (1-6): 

if "%mood%"=="1" (
    color 1F
    echo Mood set to Chill 😎
    pause
    goto MENU
)
if "%mood%"=="2" (
    color 4E
    echo Mood set to Focused 🔥
    pause
    goto MENU
)
if "%mood%"=="3" (
    color D3
    echo Mood set to Creative 🌈
    pause
    goto MENU
)
if "%mood%"=="4" (
    color 0A
    echo Mood set to Mysterious 🕵️
    pause
    goto MENU
)
if "%mood%"=="5" (
    color 07
    echo Mood reset to Default 🖤
    pause
    goto MENU
)
if "%mood%"=="6" (
    echo Goodbye! Stay Tech Savvy 💡
    exit
)

echo Invalid choice. Please try again.
pause
goto MENU

🎮 What the Colors Mean

  • 1F → Blue background, bright white text (Chill)
  • 4E → Red background, yellow text (Focused)
  • D3 → Light purple background, aqua text (Creative)
  • 0A → Black background, green text (Mysterious)
  • 07 → Default CMD look (Reset)

Change the moods and emojis to what you like.If it glitches, try removing any emojis or non-English characters.If it doesn't display anything,try running Set-ExecutionPolicy RemoteSigned to allow custom scripts.
